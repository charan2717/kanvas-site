Powerful Features

Kanvas Capabilities

Everything you need to design, deploy, and manage cloud native infrastructure with confidence

390+ Built-in Integrations

Support for all your cloud native infrastructure and applications. Kubernetes, Helm, Docker Compose, and more.

GitOps Integration

Integrate visual reviews into your CI/CD pipeline. Version control your infrastructure designs alongside your code.

Interactive Terminal

Establish live terminal sessions with one or more pods at a time. Debug and troubleshoot directly from the canvas.

Performance Metrics

Monitor your clusters in real-time. Set alerts and work with object-specific metrics for deep observability.

Timeline DVR

Playback service transactions. Scrub through the history of changes to your deployments like a video timeline.

Dry Run Deployments

Test your configurations before deploying. Catch errors early with validation against live clusters.

Multi-User Collaboration

Real-time multiplayer experience. Design and operate together with your team like Google Docs for infrastructure.

Application Import

Import existing Kubernetes manifests, Helm charts, or Docker Compose files. Visualize and manage existing infrastructure.

Component Catalog

Access thousands of versioned Kubernetes components. Build and share reusable infrastructure patterns.
